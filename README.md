This Git repository is essentially a release of source code that has been circulating for a while. It’s important to understand the context behind it, especially regarding the person known as “CIDS,” who is associated with selling something called “CidsHook.”

From what has been observed by many in the community, CIDS does not actually develop his own code. Instead, he relies heavily on work created by other developers. Large portions of the code he distributes appear to be directly copied or reused (“pasted”) from publicly available or privately shared sources. After that, the code is repackaged and sold as if it were original work.

This raises serious concerns about credibility and merit. Development, especially in technical communities, is built on skill, learning, and contribution. When someone takes existing work from others and monetizes it without meaningful modification or proper credit, it undermines the efforts of the original creators.

Multiple people have pointed out that the actual development behind these tools is done by different individuals, not by CIDS himself. In that sense, he acts more as a reseller than a developer. Because of this, it’s fair to say that the project does not reflect his own technical abilities or effort.

Now that these sources are publicly available, anyone can access them and experiment on their own. In a way, this puts everyone on equal footing — you can explore, modify, and build just like he did.
